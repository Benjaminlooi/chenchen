## [0.1.0] - 2025-11-17

### Added
- Initial MVP release
- Multi-LLM prompt submission (ChatGPT, Gemini, Claude)
- Split-screen webview layout
- Session persistence
- Authentication detection
- Structured logging
- Dual-format logging (JSON + human-readable)

### Security
- Privacy-by-design: no credential storage
- Local-only execution
- No telemetry or tracking
