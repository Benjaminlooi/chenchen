# Changelog

All notable changes to ChenChen will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.2.0] - 2026-02-09

### Added
- Advanced anti-detection for Cloudflare bypass
- Webview refresh functionality

## [0.1.5] - 2026-02-09

### Fixed
- Linux build dependencies using official Tauri recommendations

## [0.1.4] - 2025-11-21

### Fixed
- Missing `libjavascriptcoregtk-4.1-dev` dependency for Linux builds

## [0.1.2] - 2025-11-21

### Fixed
- Vitest upgrade to v4 and compatibility issues

## [0.1.1] - 2025-11-18

### Added
- Dark mode support
- Linux webview positioning fix
- UI Redesign

## [0.1.0] - 2025-11-17

### Added
- Initial MVP release
- Multi-LLM prompt submission (ChatGPT, Gemini, Claude)
- Split-screen webview layout
- Session persistence
- Authentication detection
- Structured logging
- Dual-format logging (JSON + human-readable)

### Security
- Privacy-by-design: no credential storage
- Local-only execution
- No telemetry or tracking
